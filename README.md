# 🛒 LankaGrocery - MERN Stack E-Commerce Platform

A full-stack, modern online grocery shopping web application tailored for the Sri Lankan market. Built with the MERN stack (MongoDB, Express.js, React.js, Node.js) featuring a clean UI, secure authentication, and a robust cart system.

## 🚀 Tech Stack
* **Frontend:** React (Vite), Tailwind CSS, Zustand/Redux
* **Backend:** Node.js, Express.js, JWT Authentication
* **Database:** MongoDB, Mongoose
* **Tools:** Cloudinary (Image handling), Stripe/LankaQR (Payments)

## 📁 Project Structure
The project is strictly divided into two main directories:
* `/backend` - Contains the Express server, API routes, and database models.
* `/frontend` - Contains the React Vite application and UI components.
